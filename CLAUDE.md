# gold-direct-sound

## Installed Skills

The following [stitch-skills](https://github.com/google-labs-code/stitch-skills) are installed globally for Claude Code:

| Skill | Description |
|-------|-------------|
| `design-md` | Analyze Stitch projects and synthesize a semantic design system into DESIGN.md files |
| `enhance-prompt` | Transforms vague UI ideas into polished, Stitch-optimized prompts |
| `react:components` | Converts Stitch designs into modular Vite and React components |
| `remotion` | Generate walkthrough videos from Stitch projects using Remotion |
| `shadcn-ui` | Expert guidance for integrating and building applications with shadcn/ui |
| `stitch-design` | Unified entry point for Stitch design work — prompt enhancement, design system, screen generation |
| `stitch-loop` | Iteratively build websites using Stitch with an autonomous baton-passing loop pattern |
| `taste-design` | Semantic Design System Skill for Google Stitch with premium anti-generic UI standards |

To reinstall: `npx skills add google-labs-code/stitch-skills --global --yes`
